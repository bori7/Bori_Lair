About D & W

Dead and Wounded is an old intuitive game, engaging the intuition of both players, it also stochastic probablity predictions. which makes the game very interesting.

Although, the game is highly based on accurate guesses, there are pattern to accurate predictive deciion making.

Rules of D & W

  • Each Player must pick 4 different single digits numbers from 1 - 10, not repeating any digit
  • At every guess turn, you trial must contain 4 non-repeating single digits.
  • In a guess trial, a digit is dead only if it at the right position in the original secret digits, else it is merely wounded if it only occurs in the ecret digits but not in the right spot.
  • The players announce each others result at each trials.

Play with the computer if you are a beginner.
